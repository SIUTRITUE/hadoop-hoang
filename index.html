🚀 Hadoop-Spark Cluster

Big Data Processing Platform on Docker Compose

⚡ Quick Start

  1. Run start-cluster.bat to start the cluster
  2. Wait 5-10 minutes for all services to be ready
  3. Open the Web UIs below in your browser
  4. Access Jupyter Notebook to run examples
  5. Run stop-cluster.bat to stop the cluster

🎯 Common Commands
Start Cluster:
start-cluster.bat
Stop Cluster:
stop-cluster.bat
View Logs:
docker-compose logs -f
SSH to Hadoop Master:
docker exec -it hadoop-master bash
List HDFS Files:
docker exec hadoop-master hdfs dfs -ls /
SSH to Jupyter:
docker exec -it jupyter-notebook bash
