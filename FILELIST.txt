🎯 HOW TO USE THIS PROJECT

1. READ
   → Open START-HERE.txt (you're here!)
   → Read README.md for complete guide

2. SETUP
   → Run start-cluster.bat (Windows) or ./start-cluster.sh (Linux/Mac)
   → Wait 5-10 minutes for services

3. VERIFY
   → Open http://localhost:9870 (Hadoop)
   → Open http://localhost:8080 (Spark)
   → Open http://localhost:8888 (Jupyter)

4. EXPLORE
   → Open Jupyter notebooks
   → Run examples
   → Create your own jobs

5. STOP
   → Run stop-cluster.bat (Windows) or ./stop-cluster.sh (Linux/Mac)
